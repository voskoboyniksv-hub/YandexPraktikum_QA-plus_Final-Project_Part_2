# YandexPraktikum_QA-plus_Final-Project_Part_2
SQL &amp; Autotest (VS Code, Python)
Автоматизированный тест для проверки API сервиса "Яндекс Самокат".
Тест создает заказ и проверяет, что по треку можно получить данные о заказе.

СТРУКТУРА ПРОЕКТА
-----------------
configuration.py              - файл с адресом тестового стенда
data.py                       - тестовые данные для создания заказа
sender_stand_request.py       - функции для отправки запросов к API
create_order_recive_order_by_track_test.py - основной файл с тестом
readme.md                     - этот файл с описанием
.gitignore                    - список файлов и папок, исключаемых из Git

ЧТО ДЕЛАЕТ ТЕСТ
----------------
1. Отправляет POST-запрос на создание заказа
2. Проверяет, что заказ создан успешно
3. Сохраняет номер трека из ответа
4. Отправляет GET-запрос для получения заказа по треку
5. Проверяет, что код ответа равен 200

ОЖИДАЕМЫЙ РЕЗУЛЬТАТ
-------------------
- Заказ успешно создаётся (код 201)
- Трек заказа сохраняется
- Заказ успешно находится по треку (код 200)
- В консоли появляется сообщение "Тест пройден успешно!"
